основные файлы:
1. special_matrices.c - основная программа с исследованием:
   - четыре типа матриц: плотные, треугольные, ленточные, разреженные
   - три типа распределения итераций: static, dynamic, guided
   - сравнение производительности для разных комбинаций

2. test_sizes.sh - скрипт для исследования зависимости от размера матрицы
3. test_threads.sh - скрипт для исследования зависимости от количества потоков

порядок выполнения:

1. компиляция программы:
   gcc -fopenmp -o special_matrices special_matrices.c -lm

2. запуск тестов с разными размерами матриц (4 потока):
   ./test_sizes.sh

3. запуск тестов с разным количеством потоков (матрица 2000x2000):
   ./test_threads.sh

особенности исследования:

типы матриц:
- DENSE: плотная матрица (все элементы ненулевые) - равномерная нагрузка
- TRIANGULAR: верхняя треугольная - нагрузка увеличивается к концу матрицы
- BANDED: ленточная - нагрузка зависит от положения строки
- SPARSE: разреженная - случайная неравномерная нагрузка

типы распределения:
- STATIC: равные блоки итераций - эффективно при равномерной нагрузке
- DYNAMIC: динамическое распределение порциями - эффективно при неравномерной нагрузке
- GUIDED: уменьшающиеся порции - компромисс между static и dynamic

цель эксперимента:
- исследовать влияние типа матрицы на эффективность параллелизма
- сравнить разные стратегии распределения итераций
- выявить оптимальные настройки для разных типов матриц
