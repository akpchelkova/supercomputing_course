основная программа:
- dot_product.c - mpi-программа для вычисления скалярного произведения двух векторов.
  проводит серию экспериментов с разным количеством процессов и разными размерами векторов,
  измеряя время выполнения для каждой комбинации параметров.

вспомогательные скрипты для запуска на кластере:
- job.sh - основной скрипт для запуска программы с 8 процессами на 1 узле
- test_processes.sh - комплексный тест производительности: перебирает все комбинации 
  количеств процессов (1, 2, 4, 8, 16) и размеров векторов (1k, 10k, 100k, 1M элементов)

алгоритм работы:

1. компиляция программы:
   mpicc dot_product.c -o dot_product -lm

2. запуск тестов:
   - базовый тест: sbatch job.sh
   - комплексный тест всех комбинаций: sbatch test_processes.sh

3. как работает программа скалярного произведения:
   - процесс 0 генерирует два случайных вектора заданного размера
   - векторы разбиваются на соответствующие части и распределяются между процессами
   - каждый процесс вычисляет локальное скалярное произведение для своих частей векторов
   - с помощью mpi_reduce с операцией MPI_SUM все локальные произведения суммируются в глобальное
   - измеряется и выводится время выполнения для каждой конфигурации

особенности реализации:
- используется разделение коммуникатора для тестирования разного количества процессов
- данные распределяются с учетом остатка для равномерной нагрузки
- для генерации векторов используются разные seed чтобы обеспечить случайность
- применяется операция MPI_SUM для корректного сложения частичных результатов

получаемые данные:
- время выполнения для всех комбинаций (процессы × размер_вектора)
- данные позволяют анализировать эффективность распараллеливания
- показывают как время выполнения зависит от количества процессов и объема данных
- демонстрируют масштабируемость алгоритма при увеличении вычислительных ресурсов
